# 🐍 Python Basic Projects

A collection of beginner-friendly Python projects developed to strengthen core programming concepts such as variables, loops, functions, conditional statements, file handling, dictionaries, and object-oriented programming.

## 📂 Projects

### 1. Student Management System
- Add student records
- View all students
- Search students
- Update student details
- Delete student records
- Calculate grades

### 2. Library Management System
- Add books
- View available books
- Search books
- Issue books
- Return books

### 3. Inventory Management System
- Add products
- Update stock
- Delete products
- Search inventory
- Display inventory details

### 4. Quiz System
- Multiple-choice quiz
- Score calculation
- Result summary

### 5. Expense Tracker
- Add expenses
- View expense history
- Categorize expenses
- Generate expense summary

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code
- Git
- GitHub

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Vaishnavi-Rathi-123/Python-Basic-Projects.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Python-Basic-Projects
   ```

3. Run any project:
   ```bash
   python student_management.py
   ```

   Replace `student_management.py` with the desired Python file.
